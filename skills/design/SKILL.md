# Frontend Design Skill

## Visual Hierarchy
- Use font weight and size to guide the eye.
- Maintain consistent spacing (use a 4px or 8px grid).
- Ensure sufficient color contrast for accessibility.

## Component Structure
- Keep components small and focused.
- Use composition over inheritance.
- Pass data down via props, emit events up.

## Tailwind Best Practices
- Use utility classes for layout and spacing.
- Extract repeated patterns into components, not CSS classes.
- Use responsive modifiers (sm:, md:, lg:) early.

## Accessibility (a11y)
- Use semantic HTML (<main>, <nav>, <button>).
- All images must have alt text.
- Interactive elements must be keyboard navigable.
